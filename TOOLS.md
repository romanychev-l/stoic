# Инструменты

Список того, что может помочь держать практику. Ничего обязательного — можно без всего.

## Дневник

- **Obsidian** — для дневника и заметок по ходу практики.

## Заметки на полях

Сюда можно дописывать находки по ходу практики — что сработало, что нет, какие книги читались, какие приложения помогли или наоборот мешали.
